WastingMoves supports a unique flavor of markdown format.

Headings

# Heading 1
## Heading 2
### Heading 3
#### Heading 4

Text

This is a paragraph.
Line breaks need a < br > HTML tag.
This text is *bold*.
This text is ~italic~.
This text is ^larger^.

Lists

+ List Item 1
+ List Item 2
+ List Item 3

Quotes

"This a quote."
- This is a cite.

Tables

| Column 1 | Column 2 | Column 3 |
| Column 1 | Column 2 | Column 3 |

Code

`This is code.`
> This is a code block.

Use two `` before and after a multiline text block to create a markdown formated code block.

Use three ``` before and after a multiline text block to create a code block that ignores markdown.
